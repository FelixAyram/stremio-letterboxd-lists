Letterboxd Lists para Stremio

Pega URLs de listas publicas de Letterboxd. Cada lista aparece como un catalogo en Stremio con portada, descripcion, fondo y enlaces a otros addons (Torrentio, etc.) via IMDb.

URLs de instalacion
  • PC / Desktop: http://127.0.0.1:7731/manifest.json
  • Android / TV: http://TU-IP-LOCAL:7731/manifest.json

Android: misma WiFi que la PC. Ejecuta Abrir-Firewall.bat como admin.

Instalar en Stremio

1. Copia esta URL del manifest:

2. En Stremio escritorio: Addons → icono + → pega la URL → Install

O ejecuta Instalar-en-Stremio.bat (inicia servidor + abre Stremio)

O abri (solo staging web): staging.strem.io

El servidor debe estar corriendo (npm start) mientras uses el addon.
